# Nutri Health System
A database-driven web system for managing a health and nutrition center.
Handles branches, employees, customers, inventory, sales, and purchases.

## Tech Stack
- **Database:** MySQL
- **Backend:** Python
- **Frontend:** HTML, CSS

## Project Structure
nutri-health-system/
├── PythonCode.py
├── NutriHealthDatabase.sql
├── style.css
├── templates/
│   └── (HTML pages)
## How to Run
1. Import the database:
   - Open MySQL and run `NutriHealthDatabase.sql`
2. Run the backend:
3. Open your browser and navigate to the local server

## Features
- Branch & employee management
- Inventory tracking with low-stock alerts
- Sales & purchases reporting
- Customer activity tracking
- Top products analytics
- 3NF normalized database design

## Database Entities
Branches, Employees, Customers, Health Products,
Inventory, Sales, Purchases, Suppliers, User Accounts
